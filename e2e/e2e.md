# Adobe I/O Lib Files E2E tests

## Requirements

- To run the test you'll need two OpenWhisk namespaces. Please set the credentials for those in the following env
  variables:
  - `TEST_NAMESPACE_1, TEST_AUTH_1, TEST_NAMESPACE_2, TEST_AUTH_2`

## Run

`npm run e2e`
